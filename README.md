# Tamales Shop (v.2)

A mock website that sells tamales. [View Live](https://tamales-clone.herokuapp.com/).

## Technologies Used
- ReactJS
- React Slick
- -CSS3

## Features
- Mobile-first approach
- React Slick's carousel
- CSS Grid
- Responsive Design
