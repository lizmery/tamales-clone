# Tamales Shop (v.2) - *In Progress*

A mock e-commerce website that sells tamales. [View Live](https://tamales-clone.herokuapp.com/).

## Technologies Used
- ReactJS
- React Slick
- CSS3
- CommerceJS

## Features
- Mobile-first approach
- React Slick's carousel
- CSS grid
- Responsive design
- Shopping cart
- Checkout page
- Payment processing handled by Stripe
