# Tamales Shop (v.2)

A mock e-commerce website that sells tamales. [View Live](https://tamales-clone.vercel.app/).

## Technologies Used
- ReactJS
- React Slick
- CSS3
- CommerceJS
- MaterialUI

## Features
- Mobile-first approach
- React Slick's carousel
- CSS grid
- Responsive design
- Shopping cart
- Checkout page
- Payment processing handled by Stripe
